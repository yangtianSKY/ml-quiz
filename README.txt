ML 刷题 — 机器学习基础知识题库
================================

【 使用方法 】

1. 双击 index.html 文件，用浏览器打开
2. 在首页选择难度和知识点（可多选），选好题目数量
3. 点击「开始刷题」进入答题

【 题型说明 】

📝 概念选择题（4选1）
   - 阅读题目，点击正确选项，然后点「提交答案」
   - 键盘快捷键：数字键 1-4 选选项，Enter 提交

🧩 代码拼图题（代码块排序）
   - 上方「代码块池」中有多个代码块（包含正确的和迷惑的）
   - 点击正确的代码块，按顺序排列到下方「你的答案」区
   - 点击答案区的块可以退回
   - 排好后点「提交答案」
   - 系统按正确块的比例给分（部分正确也有分）

📊 结果页
   - 查看总分、等第（A/B/C/D/F）
   - 按难度和知识点的正确率统计
   - 可展开回顾每道错题的详情

【 导入自定义题目 】

1. 参考 custom-questions.json 文件中的格式
2. 编写你自己的题库 JSON 文件
3. 在软件首页点击「导入自定义题库」选择你的 JSON 文件
4. 导入的题目会追加到内置题库中（刷新页面后消失，需重新导入）

【 题库格式说明 】

详见 custom-questions.json 中的示例。关键规则：
- 概念题：type 为 "concept"，options 必须有恰好 4 个选项，correct 为正确选项索引（0-3）
- 代码拼图题：type 为 "codepuzzle"，correct_blocks 为正确代码块（按顺序），distractor_blocks 为迷惑块（2-3个）
- difficulty 必须是 "easy" / "medium" / "hard"
- topic 必须是 "基础概念" / "经典算法" / "神经网络基础"

【 常见问题 】

Q: 浏览器打开后页面空白？
A: 请使用最新版 Edge、Chrome 或 Firefox 浏览器。

Q: 题目太多/太少？
A: 在首页调整「题目数量」下拉框（10/20/30/50/全部）。

Q: 导入的题目不见了？
A: 题目数据存在浏览器内存中，刷新页面后需要重新导入。如需永久保留，请将题目添加到 ml-quiz.html 文件中的题库里。

Q: 代码块的文字太长看不全？
A: 将鼠标悬停在代码块上，浏览器会显示完整内容。

【 技术支持 】

本软件为本地 HTML 文件，不联网，不收集任何数据。
如需修改或扩展，可编辑 ml-quiz.html 中的 JavaScript 代码。
