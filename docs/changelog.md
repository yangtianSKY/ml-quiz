# 变更记录 — ML 刷题软件

## v1.0 — 2026-06-20

### 新增
- 完整的 ML 刷题软件，单 HTML 文件（232 KB）
- 200 道内置题目（120 概念选择 + 80 代码拼图）
- 覆盖 3 个知识点：基础概念（67题）、经典算法（67题）、神经网络基础（66题）
- 3 个难度等级：简单、中等、困难
- 3 个屏幕：首页（筛选+设置）、答题页、结果页
- 概念选择题：4选1，点击选择+提交
- 代码拼图题：代码块池点击排序，含迷惑项，按块比例给分
- 结果统计：得分、等第（A-F）、按难度/知识点正确率、错题回顾
- 淡蓝主题配色，响应式设计（支持 400px+ 宽度）
- 键盘快捷键：数字键 1-4 选选项，Enter 提交
- 自定义题库导入功能（JSON 文件）
- 筛选条件无结果时自动提示
- 题量自动调整（超出可用题数时）
- 完整项目文档体系（docs/ + dev-log/ + CLAUDE.md）
- 用户使用说明（README.txt）

### 技术
- 零依赖：单个 HTML 文件，内嵌 CSS + JS
- 离线可用：无 CDN、无网络请求
- 支持 Edge / Chrome / Firefox 最新版
- JS 模块化设计：DataStore / QuestionBank / UIRenderer / QuizEngine / EventHandler
