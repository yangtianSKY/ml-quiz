# 开发需求文档 — ML 刷题软件

## 项目概述

**项目名称**：ML 刷题 — 机器学习基础知识题库

**项目目标**：为编程新手构建一个轻量、直觉的机器学习刷题工具，无需安装任何软件，双击 HTML 文件即可在浏览器中使用。

**目标用户**：机器学习初学者，有一定的 Python 基础，正在学习 ML 基础概念和算法。

---

## 功能需求

### 核心功能

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| F1 | 概念选择题 | P0 | 4 选 1 选择题，考察 ML 理论知识 |
| F2 | 代码拼图题 | P0 | 将正确代码拆成多块 + 迷惑项，用户点击选择并按正确顺序排列 |
| F3 | 判分系统 | P0 | 概念题答对得分；代码拼图按正确块比例给分 |
| F4 | 结果统计 | P0 | 答题结束后展示总分、等第、按难度/知识点正确率、错题回顾 |

### 辅助功能

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| F5 | 难度筛选 | P1 | 全部 / 简单 / 中等 / 困难 |
| F6 | 知识点筛选 | P1 | 多选：基础概念 / 经典算法 / 神经网络基础 |
| F7 | 题目数量选择 | P1 | 10 / 20 / 30 / 50 / 全部 |
| F8 | 自定义题库导入 | P1 | 用户可通过 JSON 文件扩展题库 |
| F9 | 进度显示 | P2 | 进度条 + 已答/总数 + 实时得分 |
| F10 | 错题回顾 | P2 | 结果页可展开查看每道错题的详情 |
| F11 | 跳过题目 | P2 | 答题时可跳过，不计分 |
| F12 | 键盘快捷键 | P3 | 数字键 1-4 选选项，Enter 提交 |

### 题库内容

| 知识点 | 概念题 | 代码拼图题 | 合计 |
|--------|--------|------------|------|
| 基础概念 | ~40 | ~27 | ~67 |
| 经典算法 | ~40 | ~27 | ~67 |
| 神经网络基础 | ~40 | ~26 | ~66 |
| **总计** | **~120** | **~80** | **~200** |

每个知识点内按 **简单 40% / 中等 30% / 困难 30%** 分配。

---

## 非功能需求

| 编号 | 需求 | 说明 |
|------|------|------|
| NF1 | 零依赖 | 单个 HTML 文件，无外部 CSS/JS/Font 引用 |
| NF2 | 离线可用 | 不依赖任何 CDN 或网络请求 |
| NF3 | 浏览器兼容 | 支持 Edge、Chrome、Firefox 最新版 |
| NF4 | 响应式 | 最小支持 400px 宽度，推荐 800px+ |
| NF5 | 中文界面 | 所有 UI 文字使用简体中文 |
| NF6 | 启动速度 | 文件大小 < 500KB，双击秒开 |
| NF7 | 数据安全 | 所有答题数据仅存在于浏览器内存，刷新即清除 |

---

## 不做的功能（明确排除）

- 用户注册/登录
- 云端同步
- 学习进度持久化（刷新即丢失）
- 拖拽交互（使用点击替代，更简单可靠）
- 计时功能
- 语音读题
- 移动端 App
